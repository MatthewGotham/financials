# Financials

This is where I plan to collect code that I've written for financial analysis and modelling.

## Contents

- [binomial_option_pricer.py](https://github.com/MatthewGotham/financials/blob/main/binomial_option_pricer.py): Functions for pricing and analyzing European-style and American-style options, using the binomial options pricing model.
- [option_pricing_window.py](https://github.com/MatthewGotham/financials/blob/main/option_pricing_window.py): An application that produces a GUI for pricing options, estimating implied history, and getting associated Greeks. Uses the code from binomial_option_pricer.py. TO ADD: vega and rho.
